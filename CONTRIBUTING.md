# 🤝 Contributing to ShilpSutra

Thank you for your interest in contributing to **ShilpSutra**! We welcome contributions that help preserve and authenticate India's rich handicraft heritage.

---

## 📋 How to Contribute

### 1. Fork the Repository
Click the **Fork** button at the top right of this repository to create your own copy.

### 2. Clone Your Fork
```bash
git clone https://github.com/your-username/ShilpSutra.git
cd ShilpSutra
```

### 3. Create a New Branch
```bash
git checkout -b feature/your-feature-name
```

### 4. Make Your Changes
- Write clean, well-documented code
- Follow existing code style and conventions
- Test your changes thoroughly

### 5. Commit Your Changes
```bash
git add .
git commit -m "feat: brief description of your change"
```

### 6. Push and Create a Pull Request
```bash
git push origin feature/your-feature-name
```
Then open a **Pull Request** on the main repository.

---

## 📏 Guidelines

- **Code Style**: Follow the existing patterns in the codebase
- **Commits**: Use clear, descriptive commit messages
- **Pull Requests**: Provide a detailed description of what your PR does
- **Issues**: Check existing issues before creating new ones
- **Be Respectful**: Follow our code of conduct and be kind to fellow contributors

---

## 🐛 Reporting Bugs

If you find a bug, please open an issue with:
- A clear title and description
- Steps to reproduce the bug
- Expected vs actual behavior
- Screenshots (if applicable)

---

## 💡 Suggesting Features

We love feature suggestions! Please open an issue with the **enhancement** label and include:
- A clear description of the feature
- Why it would benefit ShilpSutra
- Any implementation ideas you have
